## Installation

### Frontned
- Install NPM dependencies and run `npm start` on terminal

### Backend
- Install NPM dependencies and run `npm start` on terminal

### Mobile
- Install YARN dependencies
- run `yarn react-native start` and `yarn react-native run-ios` on terminal
